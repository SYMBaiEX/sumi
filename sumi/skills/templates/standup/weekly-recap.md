# Template: Standup — Weekly Recap

A weekly rollup of async standups into a single post. Useful when a team does daily standups but also wants a Friday summary. Built from the source material of the week, not from standup posts themselves.

## Structure

```
*{{display_name}} — {{start_date}} → {{end_date}}*

*🏁 Finished this week*
• {{Concrete things the team / I closed out. Link each.}}

*🔁 Carried over*
• {{Things still in progress that were also on last week's list.}}

*🆕 New this week*
• {{Work that wasn't on Monday's radar — why it came up.}}

*🚧 Blocked*
• {{What's been stuck and for how long. Owner of the unblock.}}

*🔜 Next week's top 3*
• {{Three highest-priority items for the coming week.}}
```

## Rules

- **"Carried over" is the honest section.** Things stuck for multiple weeks deserve a flag, not a repeat.
- **"New this week" should explain itself.** If work appeared mid-week, name the trigger (customer escalation, exec ask, dep change).
- **Top 3 is a commitment, not a wishlist.** If everything's priority, the section is useless.
- **First-person or team-first voice** depending on who the user is — match their existing standup style if known from conversational context.
