# Template: Team Update — Individual Contributor

A personal update from an IC's perspective. First-person, focused on what *I* did, what I'm doing, and what I need. Assumes the reader is a manager or team lead.

## Structure

```
*{{display_name}} — {{start_date}} → {{end_date}}*

*TL;DR*
{{One sentence. The single most important thing my manager needs to know.}}

*🏆 What I shipped*
• {{Concrete thing I landed this period. Link the PR / doc / ticket.}}

*🛠️ What I'm working on*
• {{Thing in flight. Rough % complete, milestone, or ETA.}}

*🚧 What's blocking me*
• {{Blockers I need help on. Who can unstick me? If none, write "_Nothing this period_".}}

*📅 Meetings that mattered*  ← only if Calendar is a source
• {{Decision or context from a meeting I was in.}}

*📣 Notable input I received*  ← only if Gmail is a source
• {{External or cross-team input worth flagging.}}

*🙋 What I need*
• {{Decisions, reviews, intros, time from my manager or peers.}}

*🔜 Next period*
• {{Top 3 things I'll land next period.}}
```

## Rules

- **First-person voice.** "I shipped" / "I'm blocked on" — not third-person.
- **Link everything.** PRs, docs, tickets, threads. Make it easy for my manager to click through.
- **Be specific about blockers.** "Waiting on decision about X from @person" is useful; "blocked on platform team" is not.
- **Don't hide stalls.** If something slipped, say so in one honest line — don't bury it.
